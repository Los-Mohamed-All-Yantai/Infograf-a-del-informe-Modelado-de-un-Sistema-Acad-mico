Modelado de un Sistema Académico

Universidad Andina del Cusco | Ingeniería de Sistemas | 2026-I

🎯 Objetivo

Desarrollar el modelado de una base de datos para gestionar la información académica de una universidad, asegurando integridad, organización y análisis eficiente mediante Microsoft Excel y Power Pivot.

📚 Entidades del Sistema

👨‍🎓 Alumnos

Contiene datos personales, código universitario, carrera y estado académico.

👨‍🏫 Docentes

Información profesional, especialidad y asignación de cursos.

📖 Asignaturas

Detalle de cursos, créditos, horas académicas y plan curricular.

📅 Carga Académica

Relaciona docente, asignatura, semestre y sección.

📝 Notas

Registro de evaluaciones parciales, finales y condición del estudiante.

📊 Representación del Modelo

El modelo establece relaciones entre las entidades principales mediante claves primarias y foráneas, permitiendo consultas y análisis del rendimiento académico.

📈 Análisis de Rendimiento (Ejemplo)

A través de Power Pivot se pueden obtener promedios generales, comparación de notas por evaluación y análisis del desempeño estudiantil.

✅ Conclusiones

  • Se estructuró correctamente la información académica.
  • Se garantizó integridad y relación entre entidades.
  • El modelo permite análisis dinámico del rendimiento.

💡 Sugerencias

  • Agregar entidades como Matrícula y Periodo Académico.
  • Implementar el modelo en un SGBD como MySQL o PostgreSQL.
  • Automatizar reportes mediante dashboards interactivos.
